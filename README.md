# PMx Daily — Pharmacometrics Paper Digest Archive

AI-powered daily digests of pharmacometrics literature, curated for pharmacometricians and clinical pharmacologists.

Each digest covers a recently published paper with:
- A quick take and executive summary
- Methodological breakdown
- Key equations and figures
- Critical appraisal

## Browse the Archive

Visit the live site: [PMx Daily on GitHub Pages](https://mahmabdelwahab.github.io/Pmx-daily-digest/)

Papers are tagged by topic (PopPK/PKPD, methodology, AI/ML, QSP) and searchable.

## Topics Covered

Population pharmacokinetics, PKPD modeling, quantitative systems pharmacology, model-informed drug development, exposure-response analysis, Bayesian methods, machine learning in drug development, and regulatory pharmacometrics.
